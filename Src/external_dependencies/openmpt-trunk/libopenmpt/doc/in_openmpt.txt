
in_openmpt
==========

in_openmpt is a module file (https://en.wikipedia.org/wiki/Module_file) input
plugin for WinLAMP >= 2.0 (or compatible players). in_openmpt is based on
libopenmpt.


Installation
------------

"in_openmpt.dll" must be placed into the WinLAMP "Plugins" directory, and
"openmpt-mpg123.dll" must be placed into the WinLAMP directory.


See https://lib.openmpt.org/ for documentation, FAQ and other details.
