-=-=-=-=-=-=-=-=-= The definitive audio player for Windows =-=-=-=-=-=-=-=-=-

This release includes plug-ins for MP3, MP2, WAV, MOD, XM, IT, S3M, VOC, CDDA, 
WMA, AS, and MIDI. WinLAMP also supports third-party input plug-ins, and with 
them can support SID, VQF, RA, AAC, and more. 
Check http://www.winlamp.com/plugins for plug-ins that add support for more types.
